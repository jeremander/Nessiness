# Nessiness

- Use `make` to execute commands:
  - `devserver`: Serve in dev mode (hot reloading)
  - `html`: Build files locally
  - `github`: Publish to Github Pages
  - `atproto-sync`: Sync ATProto records

Update Flex theme by pulling latest from [Flex](https://github.com/alexandrevicenzi/Flex/tree/master).
